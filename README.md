# Premium React Portfolio

A modern, responsive personal portfolio built with React, Vite, Framer Motion, and Lucide Icons.

## Features
- **Modern Design**: Dark theme with glassmorphism and smooth animations.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Sections**:
  - Navbar (Sticky & Responsive)
  - Hero Section (Intro with Call-to-Action)
  - About Me (Stats & Personal Info)
  - Experience (Timeline)
  - Projects (Grid with Hover Effects)
  - Skills (Categorized with Icons)
  - Contact (Interactive Form & Social Links)
  - Footer

## Getting Started

### Local Development
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the dev server:
   ```bash
   npm run dev
   ```

## Deployment Instructions

### 1. Upload to GitHub
1. Create a new repository on GitHub.
2. Run the following commands in your project directory:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```

### 2. Deploy to Vercel
1. Go to [vercel.com](https://vercel.com) and sign in with GitHub.
2. Click **"Add New"** -> **"Project"**.
3. Import your GitHub repository.
4. Vercel will automatically detect Vite. Click **"Deploy"**.
5. Once finished, your site will be live!
